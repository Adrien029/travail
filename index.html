Comment obtenir vos clés (5 minutes) :
1. Allez sur console.firebase.google.com
2. Cliquez Créer un projet → donnez un nom → désactivez Analytics → Créer
3. Dans le projet : Build → Firestore Database → Créer une base de données
4. Choisissez Mode test (30 jours, largement suffisant) → Activer
5. Engrenage ⚙️ → Paramètres du projet → Vos applications → </> Web
6. Donnez un nom → Enregistrer → copiez les valeurs ci-dessous
Les clés sont sauvegardées dans votre localStorage,
vous n'aurez à les entrer qu'une seule fois.
